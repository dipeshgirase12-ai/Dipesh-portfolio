# DevDipesh Portfolio 🔥

A cyber-themed portfolio website built with HTML, CSS, and JavaScript.

## 🚀 Deployment

This site is deployed on **Netlify** and auto-deploys on every push to the `main` branch.

## 🛠️ Tech Stack

- HTML5
- CSS3 (Custom properties, animations)
- JavaScript (Canvas, particle systems)
- Fonts: Orbitron, Rajdhani, Fira Code

## 📁 Project Structure

```
├── index.html          # Main entry point
├── script.js           # JavaScript logic
├── style.css           # Styles
├── assets/             # Images, PDFs, certificates
├── components/         # HTML components
└── README.md
```
