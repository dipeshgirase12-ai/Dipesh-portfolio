## ⚡ Overview

**DevDipesh Portfolio** is a fully responsive, cyberpunk-themed portfolio website built with a **TRON/Matrix aesthetic**. It showcases projects, skills, certifications, and professional experience with an interactive terminal interface, dynamic animations, and real-time visual effects.

> 🖥️ **Live Demo:** [https://dipeshgirase12-ai.github.io/Dipesh-portfolio/](https://dipeshgirase12-ai.github.io/Dipesh-portfolio/)
>
> 🚀 **Deployed on:** [Vercel](https://vercel.com/dipeshgirase12-ais-projects)

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/dipeshgirase12-ai/Dipesh-portfolio.git

# Navigate to portfolio folder
cd Dipesh-portfolio/portfolio

# Open in browser (choose one)
start index.html          # Windows
open index.html           # macOS
xdg-open index.html       # Linux

# Or serve with live reload (if you have Node.js)
npx serve .
```

## 🧪 Terminal Commands

When you open the terminal interface on the site, try these:

| Command | Description |
|---------|-------------|
| `help` | List all available commands |
| `whoami` | Display developer identity |
| `skills` | Show technical skills |
| `projects` | List deployed projects |
| `contact` | Show contact information |
| `matrix` | Toggle matrix rain effect |
| `clear` | Clear terminal output |
| `theme` | Switch between light/dark |
| `date` | Show current system time |
| `credits` | Display credits |
