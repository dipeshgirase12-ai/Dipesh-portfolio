# DevDipesh Portfolio

A modern, responsive portfolio website with glassmorphism design.

## Features

- Glassmorphism navbar with backdrop blur effect
- Animated gradient hero section
- Smooth scroll navigation
- Responsive design for all devices
- Clean, modern UI/UX

## Tech Stack

- HTML5
- CSS3
- JavaScript

## Sections

- **Home** - Hero section with introduction
- **About** - About me information
- **Projects** - Showcase of your work
- **Contact** - Get in touch form

## Quick Start

1. Open `index.html` in your browser
2. Customize the content with your information
3. Add your projects to the Projects section
4. Replace `assets/resume.pdf` with your actual resume

## Usage

Replace the placeholder content with your own:
- Personal information
- Project details
- Contact information
- Profile images
